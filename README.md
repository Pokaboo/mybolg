# 学习笔记

 [![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deployed-brightgreen)](https://pokaboo.github.io/mybolg/) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) 

本仓库用于系统化记录我的编程学习笔记，涵盖前端、后端、算法、项目实战等多个方向，最终通过 GitHub Pages 部署展示，方便自己复盘回顾，也希望能为同路人提供一点参考。

 ##  🌟 仓库介绍 

- **核心目标**：沉淀学习过程中的知识点、踩坑记录、实战总结，形成结构化的学习体系。

- **部署方式**：基于 GitHub Pages 静态部署，无需后端，访问轻便。

- **内容定位**：从基础概念到实战应用，从零散知识点到完整项目复盘，持续迭代更新。
